# scripts

Scripts
=======

Installation
------------

.. code:: bash

   git clone https://github.com/terrencetec/scripts.git
   cd scripts
   ./install

Last line will create symbolic link all scripts from scripts/bin to your
$HOME/.local/bin/

Available scripts
-----------------

- ``remove-qtile-default-groups``: remove qtile group a, s, d, f, u, i, o, p
